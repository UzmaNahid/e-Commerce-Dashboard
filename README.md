# e-Commerce-Dashboard
## Objective
To analyse e-commerce VIVA CALIF Clothing performance using Excel by transforming raw data into meaningful insights through pivot tables and dashboards.
To understand customer behavior, order trends, and regional patterns that support data-driven business decisions.
